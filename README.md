# Hi, I'm Allan.

I build **multi-agent AI systems** and **full-stack web apps** — and when I'm off the clock, I reverse-engineer 40-year-old 8/16-bit games.

By day I lead a SOC team; by night I ship open-source security tooling, indie games, and experiments at the seam between LLMs and real systems.

---

## What I'm building

- **[Mini AI SOC](https://github.com/axewater/mini-ai-soc)** — Open-source, AI-powered Security Operations Center in a single Docker Compose stack. SIEM + TIP + SOAR + IDS + SSL-intercepting proxy + SSH honeypot, with AI analyst agents driving triage, enrichment, and response. 200+ detection rules, 55+ threat-intel feeds, MITRE ATT&CK navigator.
- **Alternate Reality (remake)** — Modern remake of the 1985 Atari / Amiga / C64 / Atari ST cult classic. Engine and assets rebuilt from the ground up with modern tooling.
- **[SharewareZ](https://github.com/axewater/sharewarez)** — Turns any game folder into a dynamic, searchable, shareable library. 160+ stars.
- **[mcp-vice-emu](https://github.com/axewater/mcp-vice-emu)** / **[mcp-winuae-emu](https://github.com/axewater/mcp-winuae-emu)** — MCP servers for debugging C64/C128/VIC-20 and Amiga 68k programs inside their emulators over GDB RSP. Because AI agents should be able to debug retro assembly too.
